# SelfHub - Group AutoCreate & Raw API

SelfHub は Discord 向けの自動化ツールです。Group DM 作成や Raw API 操作を簡単に行えます。コメント、待機、ループ、条件分岐、API 呼び出しのログ表示に対応。
